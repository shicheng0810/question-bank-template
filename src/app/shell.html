Local Private Extractor

本地题库提取器

这个页面只用于你电脑本地的题库生产流程：批量导入 .mhtml/.mht、截图 OCR、修正答案、去重融合、AI 转换，并导出 GitHub Pages 题库发布包。公开站点只负责做题，不包含这里的提取能力。

Import

MHTML 与旧题库导入

上面第二个上传口可直接导入过去生成过的题库 .json / .html，自动抽取其中全部题目并加入文件列表，可像普通 mhtml 一样预览 / 修改 / 导出。
提示：如果某题你当时做错了，Canvas 回顾页会显示“红 X = 你选的错项；灰色箭头 = 正确项”。本脚本会优先把“灰色箭头”判为正确答案，不会用 checked 状态误判。

OCR

截图识别

拖拽截图到这里，或直接粘贴截图（Ctrl / Cmd + V）
支持 PNG / JPG / WebP，多次追加不会覆盖已有文件。截图会进入左侧文件列表，并复用现有预览、修正、导出 JSON、导出做题单 HTML 流程。
截图识别设置（默认本地 OCR，可选 AI 修复）
默认在浏览器端本地执行 OCR；首次使用会按需加载 OCR 依赖。当前按“一张截图≈一题”处理；若自动识别不准，可在右侧预览里继续手动修正答案后导出。
OCR 语言（纯英文截图建议 eng；只有含中文时再用 eng+chi_sim）
只发送 OCR 文本和选项，不发送截图；不会改变正确答案索引。
需先运行 npm run rapidocr:server；失败时自动回退浏览器 Tesseract OCR。
需要本地 OPENAI_API_KEY；Key 不进入浏览器。
提示：当前保留所有既有功能与边界；截图识别作为新增数据源接入，不影响原有 MHTML / 已生成题库导入逻辑。
